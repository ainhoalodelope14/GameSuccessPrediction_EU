# 📊 GameSuccessPrediction_EU

Este proyecto tiene como objetivo **predecir el éxito comercial de videojuegos en Europa** utilizando **Machine Learning**, específicamente **Random Forest Regressor** y **redes neuronales**.  

Permite analizar factores como género, plataforma, editor y otras variables para estimar las ventas potenciales antes del lanzamiento de un título.

---

## 🎯 Objetivos

- Desarrollar una herramienta que ayude a estudios de videojuegos a **tomar decisiones estratégicas** basadas en datos.  
- Optimizar recursos en marketing, desarrollo y localización.  
- Explorar el comportamiento del mercado europeo a partir de videojuegos ya lanzados.

---

- Bibliotecas de Python: `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `tensorflow` (según uso)
